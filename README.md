# 🚀 NEXUS BOARD

> **A modern collaborative engineering notes and task management system built with Vanilla JavaScript.**

NEXUS BOARD is a browser-based productivity application designed for engineering teams to organize notes, tasks, alerts, and verification checklists in a clean cyberpunk-inspired interface. The application requires no backend or installation, storing all data locally while providing a responsive and intuitive user experience.

---

## ✨ Features

* 📝 Create structured Notes
* ✅ Manage To-Do tasks
* 📋 Interactive Checklists with live progress tracking
* 🚨 High-priority Alert system
* 👥 Multi-user author identification
* 🎨 Automatic avatar generation with unique colors
* 🔍 Filter notes by category
* 💾 Local Storage persistence
* ⏰ Live system clock
* 📱 Fully responsive design
* 🌐 Runs completely offline after loading

### Multi-User Collaboration

Every note contains:

* Author name
* Automatically generated avatar

### Interactive Checklists

* Click to mark tasks complete

### Local Storage

NEXUS BOARD automatically saves:

* Notes
* Tasks
* Checklists

Your data remains available even after refreshing or restarting the browser.
